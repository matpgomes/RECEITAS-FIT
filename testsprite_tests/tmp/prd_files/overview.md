# Receitas FIT - PWA Gameficado

## Visão Geral do Produto

Aplicativo PWA (Progressive Web App) focado em receitas saudáveis e práticas para pessoas que buscam uma alimentação equilibrada. O app oferece receitas com redução calórica, acompanhamento de evolução e gamificação para manter o engajamento.

## Objetivos do Produto

1. **Facilitar alimentação saudável**: Fornecer receitas práticas e saudáveis com baixo teor calórico
2. **Gamificação**: Sistema de pontos, badges e ranking para engajar usuários
3. **Acompanhamento**: Calendário de receitas, check-ins fotográficos e evolução de peso
4. **Praticidade**: Lista de compras automática com IA para facilitar preparação

## Público-Alvo

- Pessoas buscando perder peso de forma saudável
- Usuários que precisam de praticidade no planejamento de refeições
- Público de 25-55 anos, predominantemente feminino
- Pessoas com rotina corrida que buscam facilidade

## Proposta de Valor

- Receitas testadas com comparação "normal vs fit"
- Sistema de trial de 7 dias para experimentar
- Gamificação que incentiva consistência
- Lista de compras inteligente gerada por IA
- Acompanhamento de evolução com métricas de saúde
