# DevContainer Configuration

Esta configuração permite que você desenvolva este projeto usando GitHub Codespaces ou VS Code Dev Containers.

## Recursos incluídos:

- Node.js 20
- Git
- Extensões VS Code para desenvolvimento Next.js/React
- Auto-formatação com Prettier
- ESLint configurado
- Tailwind CSS IntelliSense
- Portas 3000 e 3001 pré-configuradas para o servidor Next.js

## Como usar:

### No GitHub Codespaces:
1. Vá para o repositório no GitHub
2. Clique no botão "Code"
3. Selecione a aba "Codespaces"
4. Clique em "Create codespace on main"

### No VS Code local:
1. Instale a extensão "Dev Containers"
2. Abra o projeto no VS Code
3. Pressione F1 e selecione "Dev Containers: Reopen in Container"

O ambiente será configurado automaticamente com todas as dependências!
